# Program Description: The program sells a limited number of cinema tickets, allowing each buyer to purchase up to 4 tickets. It tracks the remaining tickets and counts the total number of buyers until all tickets are sold.

## Function: ask_for_tickets
Prompts the user to enter the number of tickets they want to buy from 1-4.

Parameters:
    None

Variables:
    tickets(int): stores the number of tickets the user wants.

Logic:
    1. prompt the user to enter the a number from 1-4.
    2. Convert the input to an integer.
    3. Return the number of tickets remaining.

Return:
    tickets(int): the number of tickets the user wants.
